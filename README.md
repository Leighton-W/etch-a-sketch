# Etch-a-Sketch

A browser-based drawing application built using HTML, CSS and JavaScript.
The project allows users to draw by moving their cursor across a dynamically generated grid and includes functionality to reset the canvas and create grids of different sizes.

## Features

* Interactive drawing grid
* Mouseover drawing functionality
* Dynamically generated grid squares
* User-defined grid size
* Grid reset functionality
* Input validation for grid dimensions
* Responsive grid generation up to 100x100

## Technologies Used

* HTML
* CSS
* JavaScript

## What I Learned

Through this project I developed my understanding of DOM manipulation, event listeners and dynamic element creation.
I gained experience creating and styling elements entirely through JavaScript, handling user input through prompts and validation, and updating the page in response to user actions. The project also helped reinforce my understanding of loops, functions and application state management.
